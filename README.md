# Blinkit-Sales Analysis - Power Bi Dashboard
## 📌 Project Objective

The objective of this project is to analyze Blinkit grocery sales data and build an interactive Power BI dashboard that provides insights into:

- Overall sales performance  
- Outlet performance by location, size, and type  
- Product category contribution  
- Customer rating trends  
- Fat content distribution across items  

The goal is to help business stakeholders make data-driven decisions to improve sales, optimize inventory, and identify high-performing outlets.

🧠 Business Insights Derived

🔹 Medium-sized outlets generate the highest revenue share.

🔹 Tier 3 locations outperform Tier 1 and Tier 2 in total sales.

🔹 Supermarket Type 1 contributes the highest number of items sold.

🔹 Regular fat products contribute more to overall sales compared to Low Fat.

🔹 Certain categories like Fruits and Snacks dominate revenue generation.
